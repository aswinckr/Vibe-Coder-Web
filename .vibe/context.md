# What We're Building

**App Name:** Discussion Prompt Generator (working title)

**One-liner:** A web app that turns YouTube videos into engaging discussion prompts for content creators.

## Who It's For
Content creators who want to spark engagement and conversations around their video content.

## How It Works
1. User pastes a YouTube link
2. App automatically fetches the transcript
3. User picks the vibe (debate questions, reflection questions, or mix)
4. AI generates discussion prompts
5. User copies prompts to use in comments, community posts, etc.

## Core Features (v1)
- YouTube link input
- Auto-fetch transcript from YouTube
- Vibe selector (debate / reflection / mix)
- AI-powered prompt generation
- Copy to clipboard

## Future Features (v2+)
- Save past generations
- User accounts & history
- Custom prompt templates
- Batch process multiple videos
